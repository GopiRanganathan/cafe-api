Welcome to the Cafe & Wifi API

GET - Search Cafe by Location

http://127.0.0.1:5000/search?loc={location}


GET - Get Random Cafe

http://127.0.0.1:5000/random


GET - Get All Cafes

http://127.0.0.1:5000/all


POST - Add New Cafe

http://127.0.0.1:5000/add

    Parameters required

  • name
  • map_url
  • img_url
  • loc
  • seats
  • toilet
  • wifi
  • sockets
  • calls
  • price

PATCH - Update Cafe Price

http://127.0.0.1:5000/update-price/{id}

    Parameter required

  • price

DELETE - Delete Cafe

http://127.0.0.1:5000/report-closed/{id}

    Parameter required

  • api-key
